# 📊 Company Operations Database (SQL Project)

## 📌 Overview

This project simulates a company database system that manages:

* Employees and hierarchy
* Branch operations
* Client relationships
* Sales transactions
* Supplier management

The goal is to demonstrate SQL proficiency in:

* Database design
* Data manipulation
* Analytical queries
* Business problem solving

---

## 🧱 Database Schema

The database consists of the following tables:

* **employee** – employee records and reporting structure
* **branch** – company branches and managers
* **client** – clients assigned to branches
* **works_with** – sales transactions between employees and clients
* **branch_supplier** – suppliers supporting each branch
